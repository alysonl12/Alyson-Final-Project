Mining & Ores

Ores are an important part of Minecraft. We can make crafts and uses of them. Below here is a list of common ores.

Search for an Ore

Ore Types

Coal: Found at any Y-level, used for fuel.

Iron: Found below Y=64, used for tools and armor.

Gold: Best found below Y=32, used for tools and powered rails.

Diamond: Found between Y=16 and Y=1, used for the best and strong tools.

Redstone: Found below Y=16, used for powering mechanisms and circuits.

Lapis Lazuli: Found below Y=32, used for enchantments and dyes.

Emerald: Found in mountain biomes, used for trading with villagers.

Best Mining Strategies

  • Strip Mining: Dig long tunnels at optimal ore levels.
  • Branch Mining: Create side tunnels off a main shaft to maximize exposure.
  • Caving: Exploring naturally generated caves for exposed ores.
  • TNT Mining: Using TNT to blow up to find exposed ores.
